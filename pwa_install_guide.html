01
GitHubアカウントを作成する
約5分 ／ 無料・メールアドレスがあればOK
  • 1github.com をブラウザで開きます
  • 2右上の 「Sign up」 をクリックします
  • 3メールアドレス・パスワード・ユーザー名を入力して登録します
    ※ユーザー名はURLに使われます。例：irohad-ecor → irohad-ecor.github.io
  • 4登録したメールアドレスに確認メールが届くので、リンクをクリックします
02
リポジトリを作成してファイルをアップロードする
約15分
  • 1ログイン後、右上の 「＋」→「New repository」 をクリックします
  • 2Repository name に irohad-app と入力します
    ※この名前がURLになります
  • 3「Public」 を選択します（GitHub Pagesの無料プランに必要）
  • 4「Create repository」 をクリックします
  • 5作成されたページで 「uploading an existing file」 というリンクをクリックします
  • 6ファイル一覧にある 全ファイルをドラッグ＆ドロップ でアップロードします
    ※iconsフォルダごとドラッグしてください
  • 7ページ下部の 「Commit changes」 をクリックします
ℹiconsフォルダをアップロードする際、フォルダごとドラッグしても認識しない場合があります。その場合は「Add file → Create new file」で icons/ と入力してフォルダを作成してから、icon-192.pngとicon-512.pngをアップロードしてください。
03
GitHub Pagesを有効にしてURLを取得する
約5分
  • 1リポジトリのページ上部にある 「Settings」 タブをクリックします
  • 2左メニューの 「Pages」 をクリックします
  • 3「Source」の項目で 「Deploy from a branch」 を選択します
  • 4Branch を 「main」「/(root)」 に設定して 「Save」 をクリックします
  • 51〜2分待つと、以下の形式でURLが表示されます：
https://（あなたのユーザー名）.github.io/irohad-app/
✓このURLをブラウザで開くとアプリが表示されます。このURLをブックマークして社内スタッフに共有してください。
✦
スマホのホーム画面にインストールする
各自のスマホで行う・1分で完了

上記のURLをスマホのブラウザで開き、以下の手順でホーム画面に追加します。

📱 iPhone / iPad（Safari）
1Safariでアプリのページを開きます
2画面下部の 「共有」ボタン（四角から上矢印のアイコン） をタップします
3「ホーム画面に追加」 をタップします
4名前を確認して 「追加」 をタップします
5ホーム画面に IROHADのアイコン が表示されます ✓
🤖 Android（Chrome）
Chromeでページを開く → 右上の「⋮」メニュー → 「アプリをインストール」または「ホーム画面に追加」をタップ → 「インストール」をタップ
✓
完了チェックリスト
✦URLは永久に無料で使えます。ファイルを更新したい場合はGitHubにファイルを再アップロードするだけで自動的に反映されます（数分で更新）。
